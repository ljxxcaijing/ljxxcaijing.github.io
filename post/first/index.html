10个步骤使用Hugo搭建github博客

Hello, Hugo!

历经一个星期，博客终于出来了。
首先撒花～

先介绍一下Hugo吧
1.Hugo is a general-purpose website framework.
2.Hugo是由Go语言实现的静态网站生成器。简单、易用、高效、易扩展、快速部署。
3.Hugo两大特点就是：“简单”和“快速”。

用Hugo搭blog步骤如下：

  1. 在Github上建一个仓库，并命名为“你的用户名.github.io”

  2. Mac中直接使用 Homebrew 安装Hugo,直接在终端里面输入

    $ brew new Hugo

  3. 使用Hugo建立网站,在我们想要搭建博客的目录下创建新的 hugo 站点

    $ mkdir blog
$ hugo new site blog/
    这样就把Hugo的相关文件建立在了blog目录中

  4. 进入blog目录，看一下其中的内容

    $ cd blog
$ ls

可以看到

▸ archetypes/
 ▸ content/
 ▸ layouts/
 ▸ static/
   config.toml

其中config.toml是网站的配置文件，包括baseurl, title, copyright等网站参数。
打开config.toml，修改其中的baseurl

 baseurl = “http://ljxxcaijing.github.io"
这样就可以连接到github上面的页面，一会儿再做相关的完善

  1. 然后开始配置主题
    Hugo有很多模版，我选择的模版是 hugo-multi-bootswatch（不过我不是很喜欢这个主题，所以后来又换了别的）
    先创建一个themes的文件夹，然后把主题有关的文件clone进去

    $ cd blog
$ mkdir themes
$ cd themes
$ git clone https://github.com/mpas/hugo-multi-bootswatch.git –depth=1

  2. 主题下载好了以后，再打开config.toml完善配置
    例如： 

    # 基本信息

baseurl = “http://ihuangmx.github.io" languageCode = “en-us” title = “Never But Life” theme =“hugo-multi-bootswatch”

[params]

github = “https://github.com/ihuangmx"

[params.strings]

# 导航标题
home_navbar_link = "首页"
blog_navbar_link = "文章"

date_format = '2006.01.02'
posts_list_header = "文章列表"

[params.theme]

inverse = true
name = "flatly"

主题选择好以后，我们可以先在本地运行一下blog，看一下效果。 在blog目录中运行： 

$ hugo server –theme=hugo-multi-bootswatch –buildDrafts
然后根据提示，在浏览器中打开 
 http://localhost:1313
即可预览blog的效果

  1. 一个blog还需要一些必备特殊的功能，首先就是代码高亮
    使代码高亮有多种方法，我选择的是安装一个highlight.js插件
    使用npm下载安装highlight.js
    $ npm install highlight.js

  1. 然后就可以开始写博文啦
    使用Hugo新建名为 first.md 的markdown文档,放到 post 目录
    以后写文章也是要在post目录中新建 

    $ hugo new post/first.md
    编辑first.md
    以下内容为一般模版：
    
+++
title         = “文章标题”
tags         = [“标签1”, “标签2”]
categories    = [“分类1”,“分类2”]
date        = “2015-01-01”
+++
正文，你好 Hugo ！

  2. 最后一步是blog成功的关键：部署！
    也就是把相关内容push到github上面去
    首先在blog目录下部署，通过命令来生成 public 目录，然后将public部署到服务器，同时设置了URL

$ hugo –theme= –baseUrl=hugo-multi-bootswatch “http://ljxxcaijing.github.io/"
然后把public目录push到github上面

$ cd public
$ git init  // 对 public 开始进行 git 管理
$ git remote add origin https://github.com/ljxxcaijing/ljxxcaijing.github.io.git // 添加远程仓库
$ git remote rm origin //如果你手抖不小心添加错了仓库，可以输入这一行来删除原来的仓库，添加正确的仓库
$ git add -A  // 跟踪所有文件
$ git commit -m “first commit” // 提交
$ git push -u origin master // 推送到远程仓库

public的文件成功push到Github上以后，你就可以访问 https://ljxxcaijing.github.io 看到自己的blog啦

  1. 之后如果写了新的文章，可以在blog目录下通过以下步骤更新 
    $ hugo // 更新 public
$ cd public
$ git add -A
$ git commit -m “文章更新”
$ git push -u origin master

    不过不要着急，过两分钟再去刷新你的blog，虽然Hugo很快，但还是做不到几秒钟就能完成部署和更新
