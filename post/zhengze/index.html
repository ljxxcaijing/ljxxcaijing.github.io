Python正则表达式与爬虫

Python正则表达式总结

当初学Python的时候，看完了基础部分就去看Flask，所以回过头来看正则表达式，其实是因为爬虫应用中，要用正则表达式来做选择和匹配。所以完整的看了正则表达式。

总结了一些常用的知识点。

正则表达式：

  • 闭包操作符

  • ｜ 任意多个

  • . 任意单个

Python正则表达式：

  • 常用is 和 is not

  • import re 导入有关正则的re模块

  • compile() 编译正则表达式

  • group() 匹配对象，没有子组返回空元组

  • groups() 匹配对象，没有自组返回全部匹配对象

  • match() 匹配字符串，结合group()显示成功的匹配

  • search() 在一个字符串中查找一个模式 匹配了返回对象，否则返回None

  • findall() 找到每个出现的匹配部分

  • sub() 搜索和替换

  • subn() 替换

  • split() 分割

EXAMPLE:

  • 使用re.match查找 – 匹配开始
>>> re.match(r'dog', 'dog cat dog')
<_sre.SRE_Match object at 0xb743e720<
>>> match = re.match(r'dog', 'dog cat dog')
>>> match.group(0)
'dog'
  • 使用re.search查找 – 匹配任意位置
search(r'cat', 'dog cat dog')
>>> match.group(0)
'cat'

>>> match = re.search(r'dog', 'dog cat dog')
>>> match.group(0)
'dog'
  • 使用 re.findall – 所有匹配对象(多)
['dog', 'dog']
>>> re.findall(r'cat', 'dog cat dog')
['cat']
