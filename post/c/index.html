在Mac上写c语言

在Mac上写c语言

其实在Mac上写c语言最好还是用Xcode吧，但是不想再装软件了，而且主要还是要用window的cFree写，要应付作业啊。

所以在Mac上就简单弄一下吧。

1.现有的软件：Sublime text 2、Vim

2.在终端里面安装gcc

3.开始编译吧！

 $ touch Hello.c
//创建一个Hello.c文件（本目录下）

 $ vim Hello.c
//用vim编辑Hello.c文件

 i
//打开vim后先输入i进入编辑状态

*********以下为vim中输入的程序*************

#include 



int main()



{



printf(“Hello Mac ! \n”);



}

 ESC键
//按ESC键退出编辑状态

:wq
//先输入“:”，再输入“w”表示保存，最后输入“q”表示退出

 $ gcc Hello.c
//编译Hello.c文件，会生成可执行文件“a.out”

 $ ./a.out
//执行a.out

然后就出现了“Hello Mac !”

成功啦！
