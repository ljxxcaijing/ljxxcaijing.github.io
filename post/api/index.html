API是什么？能做什么?

API（Application Programming Interface,应用程序编程接口）

最简单的API请求：

http://freegeoip.net/json/50.78.253.58

API和网址访问的区别：

  • API可以通过HTTP协议下载文件，几乎可以实现所有在网上干的事情
  • API请求使用非常严谨的语法
  • API使用JSON或者XML格式表示数据，而不是HTML格式

方法：

1、GET

“喂，网络服务器，请按照这个网址给我信息。” 访问http://ljxxcaijing.github.io/post/时，就会使用GET方法

2、POST

“请吧信息保存在你的数据库里。” 填写表单或者提交信息到网络服务器的后端时，就会使用POST方法

3、PUT

更新一个对象或者信息

4、DELETE

删除一个对象，在公用API里不常用

验证：

目的：“限制”用户使用API

方式：使用令牌token的方式，令牌是通过服务器对用户名和密码的组合处理后生成的

API key或者OAuth

服务器响应： API会返回格式友好的数据，数据格式大部分是JSON和XML。

XML:

<user><firstname>Ryan</firstname><lastname>Mitchell</lastname><username>Kludgist</username></user>

JSON:

{"user":{"firstname":"Ryan","Lastname":"Mitchell","username":"Kludgist"}}

API调用

URL:需要的数据范围

查询参数：过滤器或附加请求
