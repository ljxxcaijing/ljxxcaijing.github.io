Hugo搭建github的博客简易教程

Fri, Apr 22, 2016 - Read in 2 Min
Hello, Hugo! 历经一个星期，博客终于出来了。 首先撒花～ 用Hugo搭blog步骤如下： 在Github上建一个仓库，并命名为“用户名.github.io” Mac下直接使用 Homebrew 安装Hugo,直接在终端里面输入 $ brew new Hugo 使用Hugo建立网站,在我们想要搭建博客的目录下创建新的 hugo 站点 $ mkdir blog $ hugo new site blog/ 这样就把Hugo的相关文件建立在了blog文件夹中 进入blog文件夹，看一下其中的内容 $ cd blog $ ls 可以看到 ▸ archetypes/ ▸ content/ ▸ layouts/ ▸ static/ config.toml 其中config.toml是网站的配置文件，包括baseurl, title, copyright等等网站参数。 打开config.toml 修改其中的baseurl baseurl = “http://ljxxcaijing.github.io" 这样就可以连接到github上面的页面 一会儿再做相关的完善 6.然后开始配置主题 Hugo有很多模版，我选择的模版是 hugo-multi-bootswatch 先创建一个themes的文件夹，然后把主题有关的文件clone进去 $ cd blog $ mkdir themes $ cd themes $ git clone https://github.com/mpas/hugo-multi-bootswatch.git –depth=1 7.主题下载好了以后，再打开config.toml完善配置
