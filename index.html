git：简易实用功能

Thu, May 12, 2016 - Read in 1 Min
git最简单最实用的教程! 其实这主要是写给我自己看的，有的时候会忘记一些步骤，然后就一脸懵比，写博客有助于记忆。 1.github上新建一个仓库 2.本地新建一个同名的文件夹 3.在文件夹里面写你想推到github上的代码文件 4.对 public 开始进行 git 管理 $ git init 5.添加远程仓库 $ git remote add origin https://github.com/用户名/仓库名.git 如果你手抖不小心添加错了仓库，可以输入这一行来删除原来的仓库，再添加正确的仓库 $ git remote rm origin 6.跟踪所有文件 $ git add . 7.提交并加备注 $ git commit -m “first commit” 8.推送到远程仓库 $ git push -u origin master 如果报错，就先pull一下再push试试看 $ git pull origin master 等待100%出现，成功啦！
