在Mac上写c语言

Fri, Jun 3, 2016 - Read in 1 Min
在Mac上写c语言 其实在Mac上写c语言最好还是用Xcode吧，但是不想再装软件了，而且主要还是要用window的cFree写，要应付作业啊。 所以再Mac上就简单弄一下吧。 1.现有的软件：Sublime text 2、Vim 2.在终端里面安装gcc 3. $ touch Hello.c //创建一个Hello.c文件（本目录下） $ vim Hello.c //用vim编辑Hello.c文件 i //打开vim后先输入i进入编辑状态 *********以下为vim中输入的程序************* #include int main() { printf(“Hello Mac ! \n”); } ESC键 //按ESC键退出编辑状态 :wq //先输入“:”，再输入“w”表示保存，最后输入“q”表示退出 $ gcc Hello.c //编译Hello.c文件，会生成可执行文件“a.out” $ ./a.out //执行a.out 然后就出现了“Hello Mac !” 成功啦！

git：简易实用功能

Thu, May 12, 2016 - Read in 1 Min
git最简单最实用的教程! 其实这主要是写给我自己看的，有的时候会忘记一些步骤，然后就一脸懵比，写博客有助于记忆。 1.github上新建一个仓库 2.本地新建一个同名的文件夹 3.在文件夹里面写你想推到github上的代码文件 4.对 public 开始进行 git 管理 $ git init 5.添加远程仓库 $ git remote add origin https://github.com/用户名/仓库名.git 如果你手抖不小心添加错了仓库，可以输入这一行来删除原来的仓库，再添加正确的仓库 $ git remote rm origin 6.跟踪所有文件 $ git add . 7.提交并加备注 $ git commit -m “first commit” 8.推送到远程仓库 $ git push -u origin master 如果报错，就先pull一下再push试试看 $ git pull origin master 等待100%出现，成功啦！

10个步骤使用Hugo搭建github博客

Sun, Apr 24, 2016 - Read in 2 Min
Hello, Hugo! 历经一个星期，博客终于出来了。 首先撒花～ 先介绍一下Hugo吧 1.Hugo is a general-purpose website framework. 2.Hugo是由Go语言实现的静态网站生成器。简单、易用、高效、易扩展、快速部署。 3.Hugo两大特点就是：“简单”和“快速”。 用Hugo搭blog步骤如下： 1, 在Github上建一个仓库，并命名为“你的用户名.github.io” 2, Mac中直接使用 Homebrew 安装Hugo,直接在终端里面输入 $ brew new Hugo 3, 使用Hugo建立网站,在我们想要搭建博客的目录下创建新的 hugo 站点 $ mkdir blog $ hugo new site blog/ 这样就把Hugo的相关文件建立在了blog目录中 4, 进入blog目录，看一下其中的内容 $ cd blog $ ls 可以看到 ▸ archetypes/ ▸ content/ ▸ layouts/ ▸ static/ config.toml 其中config.toml是网站的配置文件，包括baseurl, title, copyright等网站参数。 打开config.toml，修改其中的baseurl baseurl = “http://ljxxcaijing.github.io" 这样就可以连接到github上面的页面，一会儿再做相关的完善 5, 然后开始配置主题 Hugo有很多模版，我选择的模版是 hugo-multi-bootswatch（不过我不是很喜欢这个主题，所以后来又换了别的） 先创建一个themes的文件夹，然后把主题有关的文件clone进去 $ cd blog
